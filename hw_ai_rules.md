# Создание системы правил и промт-шаблонов

## Проект и стек
Я выбрал проект с первого домашнего задания 
это бэкенд-сервис для роутинга заявок по банковским картам
Технологический стек: язык программирования Kotlin, основной фреймворк: 
Spring, дополнительные требования: Юнит-тесты с Kotest, Postgres, миграции Liquibase
Требования к стилю кода: Чистый, потоко и типо безопасный, поддерживаемый, масшатабируемый код

## Задача для агента
В качестве задачи буду решать проблему не обработанных заявок
Решение: Агент добавит механизм отправки заявок в статусе по шедулеру каждые 5 минут. 
Дополнительно Shedlock - для синхронизации между инстансами приложений
